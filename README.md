# Nouns Breeder

## What is it ?

1. Participate in governance
2. Claim $EGG
3. Breed a new Nouns

## Development

### Install dependencies

```sh
yarn
```

### Environment Setup

Copy `.env.example` to `.env` and fill in fields

### Commands

```sh
# Compile Solidity
yarn build


# Deploy (Testnet/Mainnet)
yarn deploy --network [network]
```
